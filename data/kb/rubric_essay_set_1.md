# Essay Set 1 评分细则（示例）

Essay Set 1 是议论文题材（是否应该限制青少年使用电脑），评分范围2-12分，四个维度：

- **内容（Content）**：观点是否清晰、论据是否充分具体
- **结构（Organization）**：段落安排是否合理、过渡是否自然
- **语言（Language/Style）**：句式是否多样、用词是否准确
- **语法（Conventions）**：拼写/标点/语法错误的密集程度

高分作文（10-12分）通常：观点明确、有具体例子支撑、结构完整（引言-论证-结论）、语言错误很少。

低分作文（2-4分）通常：观点模糊或跑题、缺乏具体论据、结构混乱、语法错误密集。
